❏コマンドの使い方

コマンドを登録することで様々な台帳の検索を高速に行えます。
①kintoneの検索フォームで「xxxx」を検索。
②URLをコピー
③任意のコマンドを入力し、URLをペースト
コマンドは任意の英数字を入力して下さい。
例のコマンドは「p」。記号を入れると検索がうまくいかないことがあります。
④コマンド+「半角スペース」+検索対象
例では「p 172.16.1.1.」
