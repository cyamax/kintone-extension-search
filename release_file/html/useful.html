❏便利な使い方

ショートカット検索

chromeの画面からすぐさま検索出来ます。

ショートカットキーの登録
その他ツール>拡張機能>キーボードショートカット からショートカットを登録することで、すぐに検索できるようになります。



kintone以外の検索

kintone以外にも検索を高速化が出来ます
※URLに検索文字が入るものみ

①好きな検索フォームで「xxxx」を検索＆URLコピー
今回はgoogleニュースでxxxxを検索。 コマンドは自由に登録。例では「gi」と設定
②オプションを開く
③コピーしたURLをペースト
④コマンド＋検索文字で検索可能
今回はkintone extension searchで「gi スイカ」と入力すると
スイカのニュースがすぐに検索できるようになりました。
